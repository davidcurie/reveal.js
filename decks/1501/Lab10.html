Leftover comments from Lab 9

Problems with manual ω

  • One period is not precise enough
  • 4 periods is better
  • 10 periods is better still


Problems with FFT ω

  • Many cycles needed for analysis
    • Remember to adjust sampling rate
    • Takes longer to collect data
  • Assumes period doesn't change during collection

Power fits

  • Exponent is very sensitive to errors in data
  • Errors in exponent affect leading coefficient
  • Fix: be sure data spans a wide range along x-axis


Linear fits

  • Need to plot new variables so data is linear
    • Use a linear fit on nonlinear data
  • $y = \omega^2$   $x = \frac{1}{m}$   slope $= k$
  • $y = \omega$   $x = \frac{1}{\sqrt{m}}$   slope $= \sqrt{k}$

Analysis

  • Is manual ω more reliable than FFT ω?
    • Find the spring constant from each data set
    • Compare each $k$ to the trusted value
    • Closest to known value = "most trustworthy"
